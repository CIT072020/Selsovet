Версия 2.0.0 от 10.02.2006
+Военно-учетный стол:
- в карточке человека добавлен реквизит ВУС при его включении открывается новая закладка ВУС с новыми реквизитами
- реестр военно-учетного стола - в данном реестре показывается все население у которого включен ВУС
- печать карточки первичного учета
+Добавлены параметры на закладке ВУС:
- "Участок" номер участка для карточек первичного учета с/с
"Печать карточки первичного учета полностью" (Да/Нет) если значение НЕТ, то не печатаются: место работы, адрес места жительства, состав семьи и даты сверок (т.к. должны заполняться карандашом)
+Добавлена возможность выполнения обработок ( Сервис -> Выполнить обработку)
+При заполнении паспортных данных человека после ввода даты выдачи паспорта автоматически проставляется орган выдачи из параметров
+Исправлена форма для статистики №14
+Исправлена форма жилищный фонд
+При вводе акта о смерти добавлена дата рождения матери и который по счету ребенок
+При вводе акта о рождении, фамилия ребенка переносится в фамилию родителей
+Добавлен реквизит гражданство для талонов прибытия и убытия
+В реестре населения выводиться теперь и адрес проживания
+Добавлен параметр на закладке ЗАГС: "Автонумерация актовых записей" (Да/Нет)
+Альбомный вид справок о рождении и смерти
+Состояние базы можно менять и создавать только при всех закрытых реестрах и его значение выводиться только в главном окне программы
+Печать в листке прибытия "орган выдачи паспорта"
+Листки прибытия и убытия теперь разрешено сохранять без даты составления
+Расширена графа "Особые отметки" при печати акта о рождении в альбомном виде
+Запрещено создавать состояния базы с меньшим годом, чем уже существуют
+Добавлен параметр на закладке Администрирование:
"Доступность восстановления текущего состояния" (Да/Нет)
Предназначен для восстановления ошибочных ситуаций; когда по ошибке из текущего состояния сформировано состояние на дату и существует необходимость восстановить текущее состояние из этого состояния при этом в действии "Создать состояние" появляется дополнительный пункт: "Восстановить текущее состояние"
+возможность сортировки в реестрах ЗАГС
